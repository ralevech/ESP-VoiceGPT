# ESP-VoiceGPT

**Голосовой AI-терминал на базе ESP32-S3**

Проект превращает отладочную плату ESP32-S3 в умное устройство, которым можно управлять голосом. Вы задаете вопрос вслух, а устройство выводит ответ от большой языковой модели (DeepSeek) прямо в браузер на вашем компьютере.

Этот проект демонстрирует мощь многозадачности (FreeRTOS) для одновременной работы с аудиопотоком, Wi-Fi и веб-интерфейсом на одном микроконтроллере.

## Мотивация проекта
Создан для изучения и демонстрации принципов:
- Многозадачного программирования на FreeRTOS.
- Работы с I2S-аудио (цифровые микрофоны и усилители).
- Взаимодействия с облачными LLM API (DeepSeek).
- Создания веб-интерфейсов для embedded-устройств.

## Аппаратное обеспечение
- **Плата:** ESP32-S3-DevKitC-1 (N16R8) с 16MB Flash и 8MB PSRAM.
- **Микрофон:** INMP441 (I2S-интерфейс).
- **Аудиовыход:** Усилитель MAX98357 + динамик 4Ω 3W (опционально).
- **Периферия:** Тактовая кнопка, макетная плата, соединительные провода.

## Архитектура RTOS (FreeRTOS)
В проекте используется двухъядерный процессор ESP32-S3 для параллельного выполнения задач. Система спроектирована так, чтобы ни одна операция не блокировала другую.

### Ядро 0 (Ввод/Вывод и Связь)
*   **Задача 1 (Audio Capture):** Высокоприоритетная задача. Чтение данных с I2S-микрофона INMP441 в кольцевой буфер.
*   **Задача 5 (WiFi Manager):** Фоновая задача. Поддержание стабильного соединения с Wi-Fi сетью.

### Ядро 1 (Обработка и Пользовательский Интерфейс)
*   **Задача 2 (Voice Activity):** Обработка аудиобуфера для определения момента, когда пользователь закончил говорить.
*   **Задача 3 (API Client):** Отправка распознанного текста в DeepSeek API и ожидание ответа.
*   **Задача 4 (WebSocket Server):** Трансляция полученного ответа во все подключенные браузеры в реальном времени.

### Межзадачное взаимодействие (Очереди)
Для обмена данными между задачами используются очереди FreeRTOS:
- `audioQueue`: Передача сырых аудиоданных из Task1 в Task2.
- `textQueue`: Передача текста запроса из Task2 в Task3.
- `responseQueue`: Передача ответа от AI из Task3 в Task4.
